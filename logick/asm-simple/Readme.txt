Элементарная реализация.
Включает в себя такие функции:
* Стрельба из пистолета
* Обработку попадания
* Ведение логов попадания
* Функции для кодирования/декодирования PWM сигналов 
  (код преобразовывает в набор "1" и "0")

Протокол построен на основе MilesTag II, но в качестве
модуляции испльзует PWM на основе (NEC) -- модуляция
тишиной, тока как MilesTag II использует модуляцию
сигналом (аналогично Sony)

Описание API можно найти в каталоге doc.
